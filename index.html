Reinforcement Learning

What is ChatMux?

ChatMux is an OpenEnv-compatible RL environment that trains AI agents to automatically prioritize chat messages by urgency. The environment simulates a busy inbox with 10-15 users sending 3-12 messages each - the agent must identify and prioritize the most urgent messages.

API Endpoints

POST /reset Start new episode
POST /step Execute action
GET /tasks List available tasks
POST /grader Grade performance
POST /baseline Run LLM baseline
GET /action-schema Get action schema

Features

• 10-15 chat users with varying relationship types
• Urgency detection using 11,578+ keywords
• Fuzzy string matching with rapidfuzz
• OpenAI API integration for LLM analysis
• 3 difficulty levels with programmatic graders
• Point-based reward system
